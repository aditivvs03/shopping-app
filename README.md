## Setup & Run

### Prerequisites
- Node.js installed
- MongoDB running locally on port 27017

### Steps

```bash
cd server
npm install
node index.js
```

Open browser at `http://localhost:3001`

---

## API Endpoints

| Method | Route               | Action         |
|--------|---------------------|----------------|
| GET    | /api/products       | Get all        |
| POST   | /api/products       | Add product    |
| PUT    | /api/products/:id   | Update product |
| DELETE | /api/products/:id   | Delete product |
